Crystal Collector!

You will be given a random number you must reach at the start of the game located in the "Score to Reach" bin.
There are four crystals below. Each crystal adds a random value between 1-12 to your current score each time it is clicked.
You win the game by matching your current score to the score to reach. You lose the game if your score goes over the score to reach.
The value of each crystal is hidden until you click on it.
Each time when the game starts, the game will change the values of each crystal.
